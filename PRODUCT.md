# Product

## Register

product

## Users

นักพัฒนาที่ทดลองสร้าง personal health tracking app สำหรับตัวเอง ใช้บน iOS/Android ดูข้อมูลสุขภาพประจำวัน

## Product Purpose

แดชบอร์ดสุขภาพส่วนตัวที่รวบรวม HRV, recovery, strain, sleep และ daily missions ในที่เดียว ให้เห็นภาพรวมสุขภาพและ coaching แบบ real-time

## Brand Personality

เรียบ, แม่นยำ, น่าเชื่อถือ — เหมือน medical instrument ไม่ใช่ fitness gamification

## Anti-references

- Colorful fitness apps ที่ฉูดฉาด (Strava, Nike Run Club)
- Hero metrics + gradient accent แบบ SaaS dashboard
- Glassmorphism cards

## Design Principles

1. ข้อมูลมาก่อน — UI ไม่แย่งความสนใจจากตัวเลข
2. Semantic color — สีแต่ละอันมีความหมาย ไม่ใช่ decoration
3. Dark by conviction — ใช้ dark theme เพราะดูตัวเลขในทุกแสง ไม่ใช่เพราะ "คูล"

## Accessibility & Inclusion

WCAG AA ขั้นต่ำ, รองรับ light/dark mode
